Плагин для вставки проигрывателя видеофайлов через shortcode

== Description ==

Плагин для вставки проигрывателя видеофайлов через shortcode. Тонкая настройка, поддержка плейлистов, возможность добавить кнопку (например, для заказа товара) прямо на видео. Отличная альтернатива стандартному плееру YouTube. Видео можно размещать на своём хостинге или Amazon S3, поддерживается формат mp4
